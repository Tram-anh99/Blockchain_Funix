Xác thực code Bước tiếp theo bạn nên làm là xác thực (validate) code để kiểm tra xem code có đang hợp lệ hay không. Bạn có thể thực hiện kiểm duyệt tự động bằng ứng dụng được cung cấp từ w3.org. Bước 1: truy cập trang https://validator.w3.org/ Bước 2: bạn có thể dùng 1 trong 3 lựa chọn: điền vào URL (nếu bạn đã đưa trang của bạn lên internet), tải lên file html hoặc copy/paste code của bạn vào ứng dụng. Bước 3: ấn CHECK và xem kết quả validate ở khung bên dưới. Bạn có thể thấy một số lỗi hiển thị ra. Bạn hãy dựa vào các lỗi và gợi ý để sửa lỗi cho trang của mình. Bạn hãy thực hiện sửa các lỗi và sau đó có thể kiểm tra lại. Hãy thực hiện để đảm bảo code của bạn không còn lỗi.
